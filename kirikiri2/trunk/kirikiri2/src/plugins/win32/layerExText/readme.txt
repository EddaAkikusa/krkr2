Title: layerExText plugin
Author: わたなべごう

●これはなに？

Layer クラスに GDI+ によるアウトラインベースのテキスト
描画拡張を行うプラグインです。

 == 特徴 ==

・プライベートフォントセットが利用できる

　アーカイブ中にあるフォントファイルを
  直接指定してフォント生成させることができます

・アウトラインベースの描画が可能

　複数の PATH/FILL を組み合わせてフォントを描画することができます

●使い方

manual.tjs 参照
環境によっては GDI+ をインストールする必要があります

●ライセンス

このプラグインのライセンスは吉里吉里本体に準拠してください。
